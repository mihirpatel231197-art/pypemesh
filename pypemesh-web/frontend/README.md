# pypemesh-web/frontend

React + Vite + TypeScript + Three.js. Deploys to Vercel.

## Dev

```bash
cd pypemesh-web/frontend
npm install
npm run dev
```

Opens at http://localhost:5173.

## Build

```bash
npm run build
npm run preview
```

## Deploy

Vercel autodetects via `vercel.json`. Root directory should be
`pypemesh-web/frontend` in the Vercel project settings.
